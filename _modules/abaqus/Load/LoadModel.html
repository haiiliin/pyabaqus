        """This method creates a BodyCharge object.

        ----------

            UNIFORM and FIELD. The default value is UNIFORM.
        field
            A String specifying the name of the AnalyticalField object associated with this load.

            empty string.

        Returns
        -------
            A BodyCharge object.
        """
